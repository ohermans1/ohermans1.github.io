A Green Energy Retailer

This website is my idea of the perfect green energy retailer (creatively called "Green Energy"), it is my most complex project yet. It is created using pure HTML and CSS (with the Sass preprocessor. No Javascript was used so extremely advanced CSS was required.

The website is fully responsive, including responsive images and responsive font-sizes (for accessibility).

This project includes a range of cutting-edge CSS techniques, including (but not limited to): clip-path, background-clip, mask-image, background-blend-mode, shape-outside, filter, backdrop-filter, object-fit, transform, column-count, hyphens, perspective, calc() and custom CSS properties.

While this website is completely coded by me, it was created during an advanced CSS course.
