A Green Energy Retailer

This website is my idea of the perfect green energy retailer (creatively called "Green Energy"), it is my most complex project yet. It is created using pure HTML and CSS (with the Sass preprocessor. No Javascript was used so extremely advanced CSS was required.

The website is fully responsive, including responsive images and responsive font-sizes (for accessibility). It also includes, but not limited to, background videos, pure css pop-up modals, advanced animations, clip-pathing and background font color clipping.

While this website is completely coded by me, it was created during an advanced CSS course.

Sassy Calculator

The creation of this simple, yet fully functional calculator combined HTML, CSS and JavaScript. CSS Grid was used to accurately layout and size the individual buttons. The calculator has a light CSS animation on mouse-hover to make it feel more interactive to the user. Carefully selected box shadows provide the illusion of the calculator floating on the page.

The calculations are carried out using JavaScript, with numerous variables and conditional statements to control the individual buttons and allow for larger numbers. The calculator screen is updated live using DOM manipulation.

JavaScript is also used to detect the device type and resize the calculator accordingly for cross-device support. calculator accordingly for cross device support.

Calculators are pretty boring, so I've also had a bit of fun with this one by using JavaScrip to give it a bit of sass, have a go!

New feature - this calculator now includes keyboard support!

Pomodoro Timer

A Pomodoro timer is a productivity-boosting tool, with which the user works for intense 25-minute chunks with 5-minute breaks. After 3 rounds of 25-minutes, the user gets a 15-minute break.

This Pomodoro timer has been developed using JavaScript, CSS and HTML.

The main features include a start, stop and pause button, a circular progress bar, colour change for each round, changeable words of encouragement and most importantly an API which delivers a dog fact from a pool of 10,000s each break! The timer also features a toggleable dark mode and an "autostart" toggle which sets whether the round starts automatically after each time-period.

Project ToDo List

This is something which I personally use on a daily basis. Created bespoke for my needs, this ramped up ToDo list dynamically grows to fit any number of projects.

Projects are time-stamped, then can be dragged and dropped between statuses, the status counters are automatically updated to reflect any changes. Projects can be created and deleted at any time!

Each new project that is created, includes a free-text note area which is automatically saved on local storage when information is added. Each time the page is loaded, it automatically checks local storage and loads any current projects you are working on.

Although defaulted to dark-mode (because that is what the end-user (me) prefers), the list can also be switched into a light mode using the toggle.

Pixel Art Maker

This fun little pixel-art maker was one of my first experiences with JavaScript and DOM manipulation.

It features a canvas size chooser, which dynamically creates the canvas based on the user entry. The user can select any colour and use it to paint on their new canvas.

As it is a "pixel art" maker, a retro-game style and background was used.
